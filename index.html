Olá! Como estudante dedicado e entusiasmado, estou ansioso para explorar novas oportunidades e estabelecer parcerias que me permitam contribuir para uma equipe dinâmica e inspiradora. Minha experiência com Python abrange desde a análise de dados até o desenvolvimento de aplicações web, demonstrando minha versatilidade como desenvolvedor. Além disso, possuo sólidos conhecimentos em HTML, CSS e JavaScript, o que me permite criar interfaces de usuário intuitivas e responsivas. Estou ansioso para trazer minha experiência e entusiasmo para novos projetos e equipes.
